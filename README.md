## Overview

Jobs99.ai is an AI-powered recruitment platform designed to help job seekers discover opportunities, optimize resumes, generate cover letters, and receive intelligent job recommendations.

The platform combines traditional job marketplace functionality with AI-assisted career tools and subscription-based premium features to streamline the job search and recruitment process.

## Local Setup

### Clone Repository

```bash
git clone https://github.com/Krunch1e/jobs99-ai.git
cd jobs99-ai
```

### Backend Setup

```bash
uvicorn backend.main:app --reload
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### AI Services

Ensure Ollama is installed and running locally if AI-powered resume processing, matching, and generation features are enabled.
